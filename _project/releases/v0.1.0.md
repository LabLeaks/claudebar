# v0.1.0 — First Public Release

Released 2026-03-28.

## What shipped

- tmux harness wrapping Claude Code with dedicated socket (`-L claudebar`)
- Status bar: peak hours, usage %, reset countdown
- Clickable menu + keyboard shortcuts for all actions
- Feature toggles with tri-state cycling (OFF / ON / ALWAYS)
- Side panes: tasks viewer, agent teams viewer, shell
- Session management: background/reattach, multi-session picker
- Restart with `--resume` on feature toggle (preserves conversation)
- Global config defaults (`~/.config/claudebar/config.json`)
- Distribution: Homebrew tap, install.sh, GoReleaser
